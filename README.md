## :fire: Iniciando a aplicação :fire:

_Abra um terminal na pasta do projeto e digite:_

Instalar dependências da aplicação:

```bash
npm i

```

Iniciar aplicação

```bash
npm run dev
```

Abra o [http://localhost:3000](http://localhost:3000) em seu navegador para ver o resultado.
